# Two Sum

## Условие задачи
Дан массив целых чисел `nums` и целое число `target`. Найдите два таких числа, чтобы их сумма равнялась `target`.

Верните индексы этих чисел в массиве (нумерация с нуля). Можно считать, что для каждого теста существует ровно одно решение. Один и тот же элемент нельзя использовать дважды.

## Формат входных данных
В первой строке записано число $n$ — количество элементов в массиве ($1 \le n \le 10^5$).

Во второй строке записаны $n$ целых чисел $a_1, a_2, \ldots, a_n$ ($-10^9 \le a_i \le 10^9$).

В третьей строке записано число $target$ ($-2 \cdot 10^9 \le target \le 2 \cdot 10^9$).

## Формат выходных данных
Выведите два числа — индексы элементов массива, сумма которых равна `target`.

## Примеры

| Входные данные | Выходные данные |
|---------------|----------------|
| 4<br>2 7 11 15<br>9 | 0 1 |
| 3<br>3 2 4<br>6 | 1 2 |
| 2<br>3 3<br>6 | 0 1 |
