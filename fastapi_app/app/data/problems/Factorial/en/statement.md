# Factorial

## Problem Statement
Find the factorial of a given number $n$.

The factorial of $n$ (denoted as $n!$) is the product of all natural numbers from 1 to $n$:

$$n! = 1 \cdot 2 \cdot 3 \cdot \ldots \cdot n$$

By definition, $0! = 1$.

## Input Format
The input file contains one integer $n$ ($0 \le n \le 20$).

## Output Format
Output one number — the factorial of $n$.

## Examples

| Input | Output |
|-------|--------|
| 0     | 1      |
| 5     | 120    |
| 10    | 3628800 |
| 20    | 2432902008176640000 |
