# A + B

## Problem Statement
Given two integers $A$ and $B$. Find their sum.

## Input Format
The input file contains two integers $A$ and $B$ ($-10^9 \le A, B \le 10^9$).

## Output Format
Output one number — the sum of $A$ and $B$.

## Examples

| Input | Output |
|-------|--------|
| 2 3   | 5      |
| -1 5  | 4      |
| 1000000000 1000000000 | 2000000000 |
